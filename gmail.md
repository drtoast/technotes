# Keyboard shortcuts

http://support.google.com/mail/bin/answer.py?hl=en&ctx=mail&answer=6594

# Navigation:

* go to all mail (g a)
* go to starred (g s)
* go to inbox (g i)
* search (/)
* compose (c)

# Conversations view:

* go to newer conversation (k)
* go to older conversation (j)
* open conversation (o)
* select conversation (x)
* move to trash (#)
* archive (e)
* star (s)

# Thread view:

* return to conversation list (u)
* reply (r), reply-all (a)
